## Project setup  

Preencher o arquivo `properties.js` somente.  

Instalar o Vue3 e o Vue Service  

```bash
npm install -g @vue/cli
```

```bash
npm i @vue/cli-service
```

## Install Dependencies  

```bash
npm install
```

### Compiles and hot-reloads for development  

Ambos

```bash
npm run serve
```

Windows

```bash
# cmd
run.bat

# PowerShell
./run.bat
```

Linux

```bash
$ ./run.bat
```
